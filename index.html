⚡ Activa la sincronización en tiempo real

Ahora mismo el horario se guarda solo en este navegador. Para que todos vean los cambios al instante, conecta Firebase Realtime Database (gratis):

  1. Ve a console.firebase.google.com e inicia sesión con tu cuenta Google.
  2. Crea un proyecto nuevo → nombre cualquiera (ej. horario-hospital).
  3. Ve a Realtime Database → Crear base de datos → modo prueba (reglas abiertas).
  4. Ve a Configuración del proyecto → Tus apps → ícono web </> → registra la app.
  5. Copia los valores del objeto firebaseConfig en la sección FIREBASE_CONFIG de este archivo index.html.
✅ Mientras tanto, la app funciona normalmente — los datos se guardan en este navegador.
